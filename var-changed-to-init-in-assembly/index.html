var block changed to init function in assembly

28 Dec 2016
  1. go

TIL - go turns a var block in init function behind the scenes.

package main

import "fmt"

var (
	testError1 = fmt.Errorf("%s ", "I am error one")
	testError2 = fmt.Errorf("%s ", "I am error two")
)

func main() {}

Pointer type in Go

Go has pointer but no pointer arithmatic. Go is strict about memory safety but sometime you need to get access to pointers. I am working on a task where I am porting some C code to Go. C code is using a lot of pointers and I wanted a way to replicate same in Go. Go has a package called unsafe and unsafe package provides a Pointer type. package main import ( "fmt" "unsafe" ) func main() { type test struct{} var myTest = new(test) myPtr := unsafe.Pointer(myTest) fmt.Printf("%T", myPtr) } myPtr is “unsafe.Pointer” type and if you want to use any operator or cast it to any other pointer, cast it to uintptr first. »

Get operating system name in Go

To get the operating system name where we running a Go program, we can use runtime package. runtime.GOOS return the operating system name. Some people try to use os.Getenv(“GOOS”) but it is not very reliable. I always use runtime package. package main import ( "fmt" "runtime" ) func main() { var goos string = runtime.GOOS fmt.Printf("The operating system is: %s\n", goos) } »
